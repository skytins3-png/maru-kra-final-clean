# MARU V13 통합 자동화 AI

오늘 대화 기준 통합본입니다. 기존 기능을 제거하지 않고 V12.3까지의 기능에 HTML 카드 코드노출 수정, 경마시간 추천없음 표시 보정 패치를 추가했습니다.

## 포함
- 경마앱/토토앱 ZIP 등록
- AI 코드생성기
- 로그 붙여넣기/로그파일 분석
- 사진 첨부/명령 입력
- 구글시트 저장
- 승인 패치
- GitHub 자동반영
- 모바일 사용
- HTML agent-card 코드노출 수정 패치
- 경마시간인데 추천 없음 표시 보정 패치
- 자동구매/자동결제 차단
